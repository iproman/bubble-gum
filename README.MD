# Bubble gum

Different tests to check your knowledge.

## Installation

```
npm i bubble-gum
```

## Run tests

```
$ npm run test:ui
```

or

```
$ npm run test:jas
```

### Addition test info

##### Run single test

```
fdescribe
```

##### Skip test

